CCGS
====

Compiling:

    1> build with scons
        $ scons         ## compile
        $ scons -c      ## clean project

    2> build with make file
        $ make          ## compile
        $ make clean    ## clean project

    3> used cmake
        $ mkdir -v build && cd build && cmake ..    ## create directory and auto-create make file
        $ make          ## compile
        $ make clean    ## clean project
